# 🏓 Pong Game (Python)

A classic Pong game built using Python's `turtle` module with object-oriented programming principles.

## 🚀 Features

* Two-player gameplay
* Real-time paddle movement
* Ball collision physics
* Score tracking system
* Increasing difficulty (ball speed increases)
* Game over detection

## 🛠️ Tech Stack

* Python
* Turtle Graphics
* Object-Oriented Programming (OOP)

## 🎮 Controls

| Player       | Keys             |
| ------------ | ---------------- |
| Left Player  | W (up), S (down) |
| Right Player | ↑ (up), ↓ (down) |

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/pong-game.git
```

2. Navigate to the project:

```bash
cd pong-game
```

3. Run the game:

```bash
python main.py
```
